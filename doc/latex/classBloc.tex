La classe \mbox{\hyperlink{classBloc}{Bloc}} représente un bloc de la grille. Cette classe permet de créer et manipuler des bloc. La classe est notamment constituée de booléens permettant de gérer les propriétés de la case\+: si elle est destructible, si elle est détruite, si une bombe a été posée dessus, si un bonus est présent sur la case... L\textquotesingle{}entier jouer\+Qui\+APosé\+La\+Bombe permet de connaître le joueur qui a posé une bombe, si une bombe a été posée.
